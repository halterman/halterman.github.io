The SGL library is relatively simple to install on all the major platforms: Microsoft Windows, Apple Mac OS X, and Linux.

2.1. Installing on Microsoft Windows XP, Vista, Windows 7, or Windows 8.1 with Visual Studio 2013

Visual Studio 2013 includes the OpenGL library, but it lacks GLUT. The GLUT library is available from http://www.xmission.com/~nate/glut.html. The zip file downloadable from this site contains the Windows binaries. The SGL library is available from http://python.cs.southern.edu/SGL. After unpacking all the files, you must copy the files to various folders within Visual Studio. Administrator privileges are required. The following instructions assume Visual Studio is installed in the default location, C:\Program Files (x86)\Microsoft Visual Studio 12.0. Adjust the instructions accordingly if Visual Studio is installed elsewhere.

  1. Create a new folder named GL: C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\include\GL. The security on the folder should be set so anyone can read its contents.

  2. Copy the files glut.h, sgl.hpp and sgl.h to the C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\include\GL folder. These files should be readable by everyone.

  3. Copy the file glut32.lib to the C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\lib folder and ensure all can read it.

  4. Copy the file glut32.dll to the C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\bin folder and ensure all can read it.

  5. Log out as administrator and log in as a normal user. You should be able to start Visual Studio and create, build, and execute an SGL program without tinkering with any project settings.
